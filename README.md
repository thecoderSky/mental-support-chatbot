# 🤖 Mental Support Chatbot

An AI-powered Mental Support Chatbot designed to assist individuals dealing with stress, anxiety, and loneliness by providing supportive and context-aware responses.

Built as part of my **Artificial Intelligence & Cloud Technology Internship (IBM SkillsBuild)**.

---

## 🚀 Overview

This chatbot focuses on delivering **empathetic and helpful responses** using a locally deployed language model, ensuring privacy and reducing dependency on external APIs.

---

## 🛠 Tech Stack

- Python  
- Ollama (Local LLM)  
- Streamlit (User Interface)  
- NLP Concepts  

---

## ✨ Features

- 💬 Context-aware conversational responses  
- 🤖 AI-powered support system  
- 🔒 Fully local execution (no external API calls)  
- ⚡ Fast response with minimal latency  
- 🖥 Simple and interactive UI using Streamlit  

---

## 🧠 Problem Statement

Most chatbot solutions:
- Depend on external APIs  
- Raise privacy concerns  
- Require constant internet connectivity  

---

## ⚙️ Solution

This chatbot:
- Uses **Ollama** to run LLM locally  
- Ensures **data privacy and offline capability**  
- Provides **real-time supportive responses**  

---

## ⚡ Key Challenges & Solutions

### 1️⃣ Generating Meaningful Responses  
**Problem:** Responses lacked context and depth  
**Solution:** Improved prompt structuring for better AI output  

---

### 2️⃣ Eliminating API Dependency  
**Problem:** External APIs caused latency and dependency  
**Solution:** Integrated locally deployed LLM using Ollama  

---

### 3️⃣ Building a Simple UI  
**Problem:** Needed a quick and user-friendly interface  
**Solution:** Used Streamlit for rapid UI development  

---

## 📚 Learning Outcomes

- Understanding of LLM integration  
- Basics of Natural Language Processing  
- Building AI-powered applications  
- Working with local AI models  
- UI development using Streamlit
